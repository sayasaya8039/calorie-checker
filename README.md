# カロリーチェッカー

品名を入力すると、その食品の平均カロリーを表示するWebアプリです。

## 機能

- 品名による食品検索（あいまい検索対応）
- 日本食を中心とした約120品目のカロリーデータベース
- ダークモード / ライトモード切り替え
- レスポンシブデザイン（スマホ対応）

## 使い方

1. 検索ボックスに品名を入力（例: カレー、ラーメン、かつ丼）
2. 該当する食品とカロリーが一覧表示されます
3. 右上のボタンでテーマを切り替えられます

## インストール・起動方法

```bash
# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev

# ビルド
npm run build

# Cloudflare Workersにデプロイ
npx wrangler deploy
```

## 技術スタック

- React 19
- TypeScript
- Vite 7
- Cloudflare Workers

## カロリーデータについて

- カロリーは1人前の平均値です
- 実際のカロリーは調理法や分量により異なります
- データは一般的な日本食を中心に収録しています
